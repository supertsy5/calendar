# Calendar

A customizable and printable calendar supporting Chinese Calendar

<https://supertsy5.github.io/calendar>

## Development

Install `trunk` before developing:
``` sh
cargo install trunk
```

### Serve
``` sh
trunk serve
```

### Build
``` sh
trunk build
```
